# Screenshot Checklist

Use this checklist when preparing the repository for GitHub visitors or recruiter review.

## Recommended Captures

- Dashboard overview with miner, monitor, and economics visible
- Terminal output showing low-impact XMRig startup
- One benchmark comparison between two CPU limit settings
- A screenshot of the local dashboard after several minutes of runtime

## File Naming

- `media/dashboard/dashboard-overview.png`
- `media/dashboard/dashboard-low-impact-run.png`
- `media/dashboard/dashboard-comparison.png`

## Tips

- Avoid exposing wallet balances you do not want public
- Avoid showing private filesystem paths if you prefer privacy
- Keep screenshots readable on GitHub without zooming
